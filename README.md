# hex2rgb

Convert HEX color to RGB

## How to use

```go
package main

import (
	hex "github.com/dlion/hex2rgb"
	"fmt"
)

func main() {
	fmt.Println(hex.Convert("#A972B6"))
}
```

Result:

```
169 114 182
```
